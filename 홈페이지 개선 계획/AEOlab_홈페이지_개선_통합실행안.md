## 2. 확정 헤드라인 (B + C + 네이버 강조 합본)

### 메인 헤드라인
> **"네이버·ChatGPT가 우리 동네에서 먼저 추천하는 가게, 누구일까요?"**

### 서브 카피 (1줄)
> **"리뷰 100개 쌓아도 AI엔 안 나옵니다 — 업종만 선택하면 30초 안에 확인됩니다"**

### 상단 배지 (위기·기회 프레임, 1차 출처 명확)
> **"네이버 검색의 40%가 AI 브리핑으로 바뀝니다 — 2026년 안에"**

**선택 근거:**
- 네이버 강조 → 한국 소상공인 최우선 채널
- 질문형(B) → 호기심 클릭 동기
- 리뷰 노력 인정(C) → 기존 행동 부정 없이 새 기회 제시
- 통계 배지(1차) → "5명 중 1명" 과장 카피 대체

## 6. 작업 시 절대 지킬 원칙

1. **PC와 모바일 별개 페이지** — `lg:`, `md:` 분기 적극 사용. 모바일 기준 iPhone 13 Pro(390×844) 검증.
2. **텍스트 크기:** `text-sm` 이상. `text-xs` 금지.
3. **반응형 패딩:** `p-4 md:p-8` 형태. `p-8` 고정 금지.
4. **이모지 1섹션 1개 이내, 전체 페이지 10개 미만.** 감정 이모지 전면 금지.
5. **기존 컴포넌트 최대 재사용.** 새 컴포넌트는 `HeroIndustryTiles.tsx` 1개만 신규.
