## 📌 개요

현재 aeolab.co.kr은 **기본적인 서비스 설명은 있으나, 기획서의 핵심 전략이 50% 이상 미반영** 상태입니다.

**목표**: Phase 0 (소상공인 30명 인터뷰) 진행 중 웹사이트가 기획서 전략을 100% 반영하도록 개선

**기대 효과**:
- 무료 체험 진입율: ~20% → ~40% (+100%)
- 무료 → 유료 전환율: ~5% → ~12% (+140%)
- 평균 체류 시간: ~45초 → ~2분 (+166%)

- 이점: 긍정 프레임 (기회) + 통계 기반 + 직접 질문으로 CTA 자연

**실행**:

- [ ] A/B 테스트 설계 (50:50 분할, 2주)
- [ ] 승자 선택 후 배포

**소요 시간**: 2시간 (라이팅 포함)

### Issue #2: 경쟁사 비교가 텍스트 리스트 형태

```

██████████████ 64 테라로사
████████ 28 내 카페 ← 격차 명확
```

**효과**: "아, 내 가게가 이렇게 뒤쳐져있구나" 즉시 직시

### Issue #4: 효과 증명 3단계 레이어 약함
**현재**:

- 레이어 2 (네이버 플레이스 신호): 페이지에 없음

**문제**: 사장님은 "점수가 올라도 손님이 올까?" 의심 (이탈 원인)

**개선안** (기획서 6.1절 참고):

#### Layer 1: AI 노출 점수
```
28점 ⬆️ +18점 → 46점 (2주 후)
= 100회 중 28회 노출 → 46회 노출
```

#### Layer 2: 네이버 플레이스 조회수
```
Before: 145명 방문
After: 312명 방문 (4주 후)
= AI 노출 증가와 함께 실제 조회수도 증가
```

#### Layer 3: 30·60·90일 Before/After 카드
```
[가입 당시 스크린샷] → [30일 후] → [60일 후] → [90일 후]
= 눈으로 직접 보는 변화
```

- [ ] 레이어 3 샘플 이미지/카드 제작 (Figma)
- [ ] "효과 증명" 섹션 전면 재설계

**소요 시간**: 2주
- 디자인: 3일

**효과**:
- 사용자 engagement 대폭 증가
- "정말 내 지역 데이터가 있을까?" 신뢰도 상승

**기술**:
- Next.js state 관리 (업종/지역 입력값 추적)
- 업종/지역별 샘플 데이터셋 준비
