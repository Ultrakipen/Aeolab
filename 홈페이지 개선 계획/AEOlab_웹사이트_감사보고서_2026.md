## 📋 Executive Summary

현재 aeolab.co.kr은 **기본적인 가치 전달 구조는 있으나**, 다음 3가지 영역에서 기획서의 핵심 전략이 약화되어 있습니다:

1. **정보 구조 문제**: 사용자가 "3초 안에 이 서비스가 뭔지" 이해하기 어려움
2. **시각적 위계 부재**: 핵심 메시지(경쟁사 비교, 효과 증명)가 본문 중간에 묻혀있음
3. **업종별 진입 전략 부재**: 음식점·카페, 미용실 등 주요 타겟별 맞춤 메시지 미구성

**평가 등급**: ⭐⭐⭐ (3/5) — 기능은 있으나 기획서 전략이 50% 이상 미반영

## 📊 1. 정보 구조 분석

### 1.1 현재 구조의 문제점

#### ❌ 문제 1: "Hero Section"에서 가치 전달 실패

**현재 메인 헤드라인**:
```
"경쟁 가게가 AI에 나오는 이유, ChatGPT한테 물어봐도 모릅니다"
```

**기획서가 원하는 메시지**:
- 명시적인 손님 손실 프레임: "손님 5명 중 1명이 이미 AI로 가게를 찾습니다"
- 직접 체험: "경쟁사는 나오는데 내 가게는 안 나온다" 확인 가능
- 즉시 행동: "지금 바로 확인하기 (1분, 회원가입 불필요)"

**실제 상황**: 
- 헤드라인이 부정적 톤 ("아, 난 모르는구나") → 문제 인식 있는 사람만 클릭
- 대다수 미인식 사용자는 다음 섹션으로 스크롤 (7초 이상 필요)
- CTA(무료 진단) 버튼 위치가 너무 아래에 있음 (스크롤 압박)

---

#### ❌ 문제 2: 단계별 체험 플로우 부재

**기획서 7.2절에서 정의한 "랜딩 체험 플로우"**:
```
① 업종·지역 입력 → ② 경쟁 가게 AI 노출 스캔 → ③ "내 가게도 확인" → ④ 무료 진단 → ⑤ 유료 전환
```

**현재 웹사이트**:
- "무료로 확인하기" 버튼 → trial 페이지 이동 (플로우 끊김)
- "결과 화면 미리보기" 링크는 있으나 **실제 데모 영상/상호작용 없음**
- 사용자가 "정말 경쟁사와 내 가게를 비교할 수 있을까?"를 직접 체험 불가

**개선 필요**:
- 랜딩 페이지 내에서 **인터랙티브 데모** (업종/지역 선택 시 실시간 샘플 스캔 결과 표시)
- "우리 동네 경쟁 현황" 섹션이 있으나, 실제 입력값 기반으로 동작하지 않음

---

#### ❌ 문제 3: 효과 증명 3단계 레이어 미약

**기획서 6.1절 효과 증명 레이어**:
| 레이어 | 지표 | 설명 | 체감도 |
|--------|------|------|--------|
| 1 | AI 노출 점수 | 100회 샘플링 기반 | ★★★☆☆ |
| 2 | 검색 유입 신호 | 네이버·구글 비즈니스 연동 | ★★★★☆ |
| 3 | Before/After 시각 증거 | 30·60·90일 스크린샷 비교 | ★★★★★ |

**현재 웹사이트**:
- 레이어 1만 강조 (AI 노출 점수 82, 73, 64 샘플)
- **레이어 2 (네이버 플레이스 조회수 변화)**: 페이지에 없음
- **레이어 3 (Before/After 카드)**: "이런 변화를 기대할 수 있습니다" 섹션에 텍스트만 있고 **시각적 카드/비교 이미지 없음**

→ 사장님은 "점수가 올라도 실제로 손님이 늘어나나?" 의심 (이탈 위험)

6. ⚠️ **70% 스크롤 후**: "ChatGPT도 방법은 알려줍니다" — AEOlab 차별화

3️⃣ "지금 바로 확인" CTA (즉시 행동 유도)
4️⃣ 효과 증명 3단계 레이어 (구독 고민할 때)
```

**현재는**: 설명을 위한 설명으로 구성되어 있음 → "뭔가 복잡한 서비스"라는 인식

## 🎯 3. 업종별 진입 전략 부재 (치명적)

기획서의 **가장 강력한 전략** 중 하나인 **"업종별 진입 언어"** (5.2~5.3절)이 웹사이트에 완전히 빠져있습니다.

### 3.1 기획서의 업종별 전략

**음식점·카페** (기획서 5.2절):
```
기존 행동 (인정): 네이버 리뷰 관리
핵심 메시지: "리뷰는 계속 하세요. 그 리뷰를 AI 브리핑에서도 읽히게 만들어드립니다."
설득 프레임: 미래 위협 / 리뷰 자산 확장 / 경쟁자 비교
```

**미용실·뷰티**:
```
기존 행동: 인스타 포트폴리오 관리
핵심 메시지: "인스타 포트폴리오 예쁘게 관리하고 계시죠. 그런데 ChatGPT에서 헤어컷 추천 물어보면 안 나옵니다."

**병원·한의원**:
```
기존 행동: 블로그·지식인 운영
핵심 메시지: "환자분들이 ChatGPT에 진료과목 잘 하는 병원 물어봅니다. 거기서 선생님 이름이 나오나요?"
```

### 3.2 현재 웹사이트의 문제

**업종 선택 드롭다운**:
```
음식·식음료 / 카페·디저트 / 뷰티·패션 / 의료·건강 / 교육·보육 / 전문직·법무·세무 / 쇼핑·유통 / 생활서비스·인테리어 / 문화·여가·취미 / 숙박·여행·이벤트 / 사진·영상촬영 / 디자인·인쇄 / IT·웹·마케팅
```

**문제**:
- 드롭다운 선택만 있고, **선택 후 맞춤 메시지 없음**
- 예: "음식점 선택" → "리뷰 자산 확장" 메시지로 전환되지 않음
- 모든 사용자에게 동일한 "경쟁사 비교 + 점수 + 가이드" 메시지

## 🚨 5. 치명적 문제 3가지

### 🔴 1. 모바일 진입자 이탈률 높음

**현상**: 
- 모바일에서 업종 드롭다운 선택 후 "무료로 확인하기" 클릭
- → trial 페이지로 이동
- → 로딩 시간 + 다시 업종·지역 입력 요청
- → **최소 30초 소요** (기획서의 "1분" 약속 미달성, 실제로는 1분 30초)

**원인**: 
- 랜딩 페이지와 trial 페이지가 분리되어 있음
- 입력값 전달이 URL 파라미터로 되지 않음

**해결**:
```
방안 A: trial 페이지로의 리다이렉트 전에 URL에 ?industry=cafe&region=gangnam 파라미터 전달
방안 B: 랜딩 페이지 내에서 결과 렌더링 (더 좋음)
```

---

### 🔴 2. 효과 증명 약함 → 구독 이탈 위험

**문제 진단**:
- 대시보드 프리뷰에서 "AI 노출 점수 67점"만 강조
- 실제 사장님이 궁금한 것: "점수가 올라도 손님이 올까?"

**기획서 6.2절의 해결책**:
> "주간 카카오톡 알림이 '이번 주 변화 없음'이면 이탈 원인이 된다"

**현재 웹사이트**:
