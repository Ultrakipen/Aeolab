> 개발 기준 파일: `backend/middleware/plan_gate.py` · `frontend/lib/plans.ts`

## 1. 서비스 개요

AEOlab은 한국 소상공인이 **네이버 AI 브리핑**과 **글로벌 AI 검색**에 더 잘 노출되도록 돕는 AI 검색 최적화 플랫폼입니다.

### 핵심 가치 제안

| 사용자 유형 | 핵심 질문 | AEOlab의 답 |
|------------|----------|-------------|
| 소상공인 | 내 가게가 네이버 AI 브리핑에 나오나? | 듀얼트랙 진단 + 성장 단계 즉시 확인 |
| 소상공인 | 경쟁 가게보다 왜 뒤처지나? | 경쟁사 키워드 갭 분석 + 바로 쓸 개선 문구 |
| 소상공인 | 오늘 당장 뭘 해야 하나? | AI 브리핑 4경로 + 복사 버튼 즉시 적용 |
| 예비 창업자 | 이 지역·업종 AI 경쟁이 얼마나 심한가? | 창업 타이밍 지수 + 선점 키워드 발굴 |
| 다점포 운영자 | 5개 매장을 한눈에 관리할 수 있나? | Biz 플랜 통합 현황 대시보드 |

### 3-1. 무료 체험 (비로그인, 0원)

> **대상:** 서비스를 처음 접하는 소상공인  
> **URL:** `/trial`  
> **제한:** 하루 3회 (IP 기준), 비로그인

#### 제공 기능

**듀얼트랙 AI 가시성 진단**
- 업종·지역·사업장명·키워드 입력 후 즉시 분석
- 스마트플레이스 현황 체크박스 (FAQ 등록 여부 / 최근 소식 여부 / 소개글 여부)
- 리뷰 텍스트 입력 → 키워드 커버리지 분석

**결과 화면 — 5개 지표 즉시 확인**

| 지표 | 설명 |
|------|------|
| Track 1 점수 (네이버 채널) | 네이버 AI 브리핑·스마트플레이스 최적화 점수 (0~100) |
| Track 2 점수 (글로벌 AI 채널) | Gemini·ChatGPT·Claude 등 글로벌 AI 노출 점수 (0~100) |
| 통합 점수 (Unified Score) | 업종별 가중치 적용 최종 점수 (restaurant 70:30 등) |
| 성장 단계 | 생존기 / 안정기 / 성장기 / 지배기 4단계 |
| 업종별 AI 채널 비중 | 네이버 vs 글로벌 AI의 업종별 중요도 설명 |

**키워드 분석 — 바로 쓸 수 있는 액션 제공**

| 항목 | 설명 |
|------|------|
| 없는 키워드 TOP 3 | 업종 내 필수 키워드 중 현재 가게에 없는 것 |
| 선점 키워드 (Pioneer) | 경쟁사도 아직 쓰지 않는 기회 키워드 (Emerald 배지) |
| FAQ 복사 버튼 | 1순위 키워드 포함 스마트플레이스 FAQ 문구 즉시 복사 |

**유료 전환 CTA**
- 회원가입 유도 배너 (Basic 플랜 비교)
- 쿨다운 카운트다운 표시 (하루 3회 초과 시 남은 시간 표시)

### 3-2. Basic 플랜 (9,900원/월)

> **대상:** AI 검색 노출을 처음 관리하기 시작하는 소상공인  
> **킬러 기능:** 매주 자동 진단 + 오늘 바로 쓸 수 있는 개선 도구

#### A. 대시보드

**듀얼트랙 AI 가시성 카드**
- 네이버 채널 점수 / 글로벌 AI 채널 점수 분리 표시
- 통합 점수 + 업종별 가중치 설명 (예: 음식점 = 네이버 70% / 글로벌 30%)
- 성장 단계 배지 (생존기→지배기)
- 없는 핵심 키워드 amber 박스 (바로 추가해야 할 키워드 목록)
- 9개 업종별 맞춤 팁 메시지

**리뷰 감정 분석 대시보드** _(신규 2026-04-14)_
- 긍정 / 중립 / 부정 비율 3색 바 차트
- 자주 등장하는 긍정·부정 키워드 배지
- ai_citations에서 발췌문 자동 수집 → Claude Haiku 배치 분석
- 1시간 TTL 캐시 적용 (API 비용 최적화)

**Before/After 히스토리 카드**
- 사업장 등록 시 자동 캡처된 Before 스크린샷
- 스캔 후 After 캡처 비교
- 30일 점수 추세선 (Recharts)

**AI 멘션 현황**
- 7개 AI 플랫폼 노출 결과 표 (Gemini·ChatGPT·Perplexity·Grok·네이버·Claude·Google)
- 플랫폼별 멘션 여부 + 발췌 문장

**사업장 등록 온보딩**
- RegisterBusinessForm: 업종·지역·키워드·Google Place ID·카카오 Place ID 입력
- 등록 즉시 Before 스크린샷 자동 캡처

#### B. 스캔 시스템

**자동 스캔 (APScheduler)**
- 매주 월요일 새벽 2시: 7개 AI 풀스캔 자동 실행
- 매일 새벽 2시: 네이버·Gemini 핵심 지표 자동 수집
- 스캔 완료 시 카카오 알림톡 발송 (설정 ON 시)

**수동 스캔**
- 하루 2회 허용 (중요한 날 즉시 확인)
- 실시간 SSE 스트림 진행 상황 표시 (ScanTrigger 컴포넌트)
- 스캔 완료 시 5초 성공 메시지

#### C. 경쟁사 관리

**경쟁사 등록 (최대 3곳)**
- 3가지 방식: 지역 검색(네이버 API) / 직접 입력 / AEOlab 내 동종업계 추천
- 경쟁사별 AI 노출 점수 비교 막대 차트

**키워드 갭 분석**
- 내가 보유한 키워드 (초록)
- 내게 부족한 키워드 (빨강)
- 경쟁사만 쓰는 키워드 (보라)
- 선점 가능 기회 키워드 (에메랄드)
- QR 카드 유도 문구 자동 생성

#### D. 개선 가이드 페이지 (Guide)

**AI 브리핑 현황 배너**
- 네이버 AI 브리핑 현재 노출 상태 요약

**성장 단계 카드**
- 현재 단계 (생존기·안정기·성장기·지배기)
- 이번 주 해야 할 것 / 하지 말 것 체크리스트

**AI 브리핑 직접 관리 4경로** _(Briefing Engine)_

| 경로 | 설명 | 소요 시간 |
|------|------|----------|
| 경로 B: FAQ 등록 | 스마트플레이스 Q&A — AI 브리핑 가장 직접적 인용 | 5분 |
| 경로 A: 리뷰 답변 | 키워드 포함 답변 초안 (긍정·부정·일반) | 3분 |
| 경로 C: 소식 업데이트 | 주 1회 소식 초안, 최신성 점수 유지 | 5분 |
| 경로 D: 소개글 수정 | 한 번만, 영구 키워드 기반 소개글 | 10분 |

**스마트플레이스 FAQ 초안 생성** _(신규 2026-04-14, 월 5회)_
- Claude Haiku로 업종별 Q&A 5개 자동 생성
- 각 항목 복사 버튼 (스마트플레이스 Q&A 탭에 바로 붙여넣기)

**선점 키워드 현황 (KeywordGap Card)**
- 보유·부족·경쟁사 전용 키워드 시각화
- QR 유도 문구 1순위 키워드 타겟팅 복사

**선점 키워드 상세 분석** _(신규 2026-04-14, Basic+)_
- "상세 분석 보기" 버튼 클릭 시 lazy load
- 키워드별: 왜 기회인지 이유 + 리뷰·소개글에 바로 쓸 예시 문장
- 2시간 TTL 캐시 적용

**리뷰 답변 초안 (월 10회)**
- 긍정·부정·일반 리뷰별 키워드 포함 초안
- 네이버 정책 준수 (혜택 제공 문구 제외)

**즉시 활용 도구 (QuickTools)**
- 리뷰 유도 문구 + QR 메시지
- 소식 초안 자동 생성
- 핵심 키워드 목록 복사

**FAQ 섹션**
- 스마트플레이스 Q&A 등록용 (접기/펼치기)
- AI 검색 최적화 FAQ

#### E. Schema JSON-LD 자동 생성

- LocalBusiness + FAQ Schema 자동 생성
- 복사 버튼 / 직접 다운로드
- Google Rich Results 최적화

#### F. 설정

- 카카오 알림 수신 설정 (스캔 완료 알림 / 경쟁사 순위변동 알림)
- 구독 플랜 확인 및 취소
- 계정 정보 관리
