## 시나리오 1 — 동네 카페 사장 박민준

### 배경
강남구 논현동 카페. 네이버 예약은 있지만 "강남 카페 추천" 검색하면 AI가 경쟁 카페만 소개해서 최근 매출이 줄었다. 지인에게 AEOlab 얘기를 들었다.

---

### 1-A. 긍정 시나리오 — "처음엔 반신반의, 결국 단골 고객 늘었다"

**1단계: 무료 체험 (비로그인)**

> *스마트폰으로 `/trial` 접속. "강남 논현동 카페 / 아메리카노 디저트 브런치" 입력*

- 결과 화면에서 **Track 1 점수 22점 (생존기)** 확인
- "네이버 AI 브리핑에 나오지 않습니다" 문구 → 충격 받음
- **없는 키워드 3개:** `브런치 카페`, `논현 디저트`, `논현 커피맛집`
- FAQ 복사 버튼 클릭 → 그 자리에서 스마트플레이스 Q&A에 붙여넣기
- 회원가입 CTA 클릭

**2단계: Basic 가입 (9,900원)**

> *신용카드 등록, 토스 자동결제 설정*

- 월요일 자동 스캔 결과 카카오 알림 수신
- 가이드 페이지에서 **AI 브리핑 4경로** 확인
  - 경로 B(FAQ) 완료 → 경로 C(소식 주 1회) 실행
- 3주 후: Track 1 점수 22 → 38점 상승 (안정기 진입)
- "브런치 카페 논현" 검색 시 네이버 AI 브리핑에 첫 노출 확인
- **리뷰 답변 초안** 기능으로 부정 리뷰 키워드 포함 답변 작성
- 스마트플레이스 FAQ 초안 5개 생성 → 전부 등록

**3단계: 6개월 후 — 안정적 이용**

- 매주 월요일 스캔 결과 확인이 루틴으로 자리잡음
- 경쟁사 3곳 등록 → "옆 카페는 '논현 라떼' 키워드를 쓰고 있구나" 파악
- **긍정 결과:**
  - Track 1 점수 22 → 61점 (성장기)
  - 네이버 AI 브리핑 노출 주 3회 이상
  - 인스타·블로그 유입보다 AI 검색 유입이 더 많아짐
  - 지인에게 추천 → 구전 효과

**핵심 성공 요인**
- FAQ 복사 버튼이 "오늘 당장 할 수 있는 것"이어서 행동력 높음
- 카카오 알림으로 결과를 잊지 않고 꾸준히 확인
- 복잡한 SEO 지식 없이도 실행 가능

---

### 1-B. 부정 시나리오 — "한 달 쓰다가 결제 취소"

**1단계: 무료 체험**

- 결과를 보긴 했는데 **숫자의 의미를 모름** ("Track 1이 뭐지?")
- FAQ 복사는 했지만 어디에 붙여넣는지 몰라서 포기
- 그냥 회원가입

**2단계: Basic 가입**

- 대시보드 접속했는데 **사업장 등록 과정이 번거롭다고 느낌**
  - 카카오 Place ID가 뭔지 몰라서 빈칸으로 제출
  - Before 스크린샷 캡처에 2~3분 소요 → "왜 이리 오래 걸려"
- 월요일 스캔 결과 카카오 알림 수신 → 점수가 낮다는 것만 확인, 뭘 해야 할지 모름
- 가이드 페이지 들어갔는데 텍스트가 많아서 **읽다가 이탈**
- AI 개선 가이드 생성 → 내용은 좋은데 Claude가 생성한 가이드를 실제로 적용하는 방법을 모름

**3단계: 1개월 후 — 이탈**

- 점수가 22 → 24점으로 거의 변하지 않음 (실제 아무 행동도 안 함)
- "9,900원이 아깝다" → 구독 취소
- **이탈 원인 분석:**
  - 가이드를 읽는 것과 실행 사이의 간격을 좁히지 못함
  - "오늘 5분 안에 할 수 있는 것 1가지"를 앱이 먼저 제안했어야 함
  - 카카오 Place ID 같은 기술적 개념에 대한 설명 부재

**개선 포인트**
- 온보딩 1단계: 사업장 등록 시 Place ID 자동 검색 기능
- 대시보드 진입 시 "오늘 할 일 1가지" 배너를 최상단에 배치
- 가이드 페이지: 긴 텍스트보다 "5분 실행 카드" 형식

## 시나리오 2 — 미용실 원장 김지연

### 배경
마포구 홍대 미용실. 블로그·인스타는 꾸준히 올리고 있지만, 최근 경쟁 미용실이 네이버 AI에서 먼저 소개된다는 고객의 말을 들었다. IT에 어느 정도 익숙하다.

---

### 2-A. 긍정 시나리오 — "Pro로 업그레이드하고 경쟁사를 앞질렀다"

**1단계: 무료 체험**

- "홍대 미용실 / 염색 펌 커트" 입력
- Track 1 점수 41점 (안정기) — "어, 생각보다 낮네"
- 없는 키워드: `홍대 탈색`, `마포구 펌 맛집`, `홍대 컬러 맛집`
- **선점 키워드:** `홍대 새치 커버` (경쟁사도 안 씀) → 흥미 유발

**2단계: Basic 가입 → 3개월 후 Pro 업그레이드**

- Basic 사용 2개월:
  - 경쟁 미용실 3곳 등록 → 상위 경쟁사가 "홍대 매직 스트레이트" 키워드를 집중 사용 중임을 발견
  - 리뷰 답변 초안으로 "염색 결과 아쉽다"는 부정 리뷰 → "다음엔 홍대 탈색 전 모발 진단을 먼저 받으시면..." 키워드 포함 답변 작성
  - AI 개선 가이드 5회 소진 → 더 많이 필요함

- **Pro 업그레이드 트리거:**
  - 경쟁사 1곳이 갑자기 AI 노출 점수가 오름 → "뭘 바꿨지?"
  - Pro의 **경쟁사 변화 즉시 알림** 기능을 알게 됨
  - 경쟁사 10곳까지 모니터링 가능 → "홍대 미용실 전체를 보고 싶다"

- Pro 2개월:
  - 경쟁사 8곳 등록 → 아무도 안 쓰는 `홍대 새치 커버` 키워드 독점
  - ChatGPT 광고 대응 가이드 활용
  - PDF 리포트 다운로드 → 배우자에게 성과 보고용으로 활용
  - **결과:** Track 1 점수 41 → 73점 (성장기 상단)

**핵심 성공 요인**
- 블로그 경험이 있어서 키워드 개념을 빠르게 이해
- 경쟁사 변화 알림이 경쟁 심리를 자극해 지속 사용 유도
- 리뷰 답변이 실제 고객 응대에 바로 활용

---

### 2-B. 부정 시나리오 — "Pro 결제했는데 경쟁사 알림이 안 온다"

**문제 상황**

- Pro 가입 후 "경쟁사가 FAQ·키워드 추가하면 즉시 알림" 기능 기대
- 그러나 카카오 알림톡 템플릿이 **아직 심사 승인 전**
  - `AEOLAB_COMP_01` 템플릿 미승인 상태 → 알림 미발송
- 1주일이 지나도 알림이 안 옴 → "사기 당한 거 아냐?"
- 경쟁사 변화가 있었는데 확인을 못함

**추가 부정 경험**

- 주 3회 자동 스캔 중 수요일 스캔이 서버 부하로 20분 지연
- "업그레이드했는데 Basic이랑 별 차이 없는 것 같다" 느낌
- PDF 다운로드 → 한글 폰트 깨짐 (NotoSansCJK 경로 설정 문제)

**이탈 경로**

- 22,900원 × 2개월 지불 후 취소
- 네이버 카페에 부정 후기 작성 위험

**개선 포인트**
- 카카오 알림 미승인 상태에서 Pro 가입 유도 시 → 대체 이메일 알림으로 임시 제공
- 수요일 스캔 실패 시 자동 재시도 + 사용자에게 안내 문자
- PDF 생성 전 로컬 폰트 경로 사전 검증

## 시나리오 3 — 예비 창업자 이승호

### 배경
홍대·신촌 라멘집 창업 준비 중. IT 지식이 있고 데이터에 기반한 의사결정을 선호. 창업 전 경쟁 분석이 필요해서 AEOlab을 찾았다.

---

### 3-A. 긍정 시나리오 — "창업 전에 선점 키워드를 잡았다"

**1단계: 무료 체험**

- "신촌 라멘" 검색 → 창업 타이밍 지수: **기회 있음 (기회지수 85점)**
  - 신촌 라멘 등록 사업장 2곳 → "경쟁이 별로 없구나"
- 글로벌 AI 채널 비중이 높음 → "라멘은 ChatGPT에서도 검색하네"

**2단계: 창업 패키지 가입 (16,900원)**

- 창업 시장 분석 리포트 생성
  - 경쟁사 2곳 AI 점수 분석
  - **틈새 키워드 발굴:** `신촌 라멘 늦은 밤`, `신촌 1인 라멘`, `연대 라멘 국물`
  - Claude의 진입 전략: "야간 영업 + 1인석 특화로 차별화, 오픈 전 FAQ 미리 등록"
- 오픈 전 스마트플레이스 등록 + FAQ 5개 미리 작성
- 소개글에 발굴한 틈새 키워드 전부 포함

**3단계: 오픈 후 결과**

- 오픈 첫 주 Track 1 점수 51점 (성장기) — 경쟁사는 35점
- "신촌 1인 라멘" 검색 → 네이버 AI 브리핑 3위 노출
- 오픈 한 달 만에 "신촌 라멘 늦은 밤" 검색 1위
- **창업 패키지 → Basic으로 전환** (창업 분석 목적 달성)

**핵심 성공 요인**
- 창업 전 선점 키워드를 스마트플레이스에 미리 심어둠
- 경쟁이 적을 때 진입 → AI 학습 기간 선점
- 데이터 기반 차별화 전략 수립

---

### 3-B. 부정 시나리오 — "분석 결과가 너무 낙관적이었다"

**문제 상황**

- 창업 타이밍 지수 "기회 있음" → 실제로 오프라인 경쟁은 매우 치열
- AEOlab 데이터는 **AEOlab에 등록된 사업장 기준** → 실제 홍대 라멘집은 50개가 넘음
- "데이터 수집 중" 지역 특성상 점수 신뢰도 낮음

**이탈 원인**

- "시스템이 홍대 라멘집 전체를 모르고 있구나" → 신뢰 하락
- 창업 리포트의 Claude 전략이 너무 일반적 ("SNS 활용하세요" 수준)
- 16,900원 값어치를 못 한다고 느낌

**개선 포인트**
- 창업 타이밍 지수에 "이 수치는 AEOlab 등록 기준입니다. 실제 경쟁은 더 많을 수 있습니다" 면책 문구 명시
- 실제 네이버 플레이스 검색 결과 수와 연동하여 경쟁 수준 보정
- Claude 전략 프롬프트에 업종별 특수성 더욱 반영

## 시나리오 4 — 다점포 운영자 최인국

### 배경
치킨 프랜차이즈 4개 매장 운영. 직원에게 맡기기가 어렵고 매장별로 AI 노출이 다른 이유를 모름. 스마트폰은 쓰지만 앱에는 익숙하지 않음.

---

### 4-A. 긍정 시나리오 — "Biz 플랜으로 4개 매장 한눈에 관리"

**1단계: 지인 소개로 유입 (광고 X)**

- 직원이 대신 무료 체험 해봄 → "사장님, 이거 우리 매장 점수 너무 낮아요"
- 직원이 회원가입 + Basic 가입

**2단계: Basic → Biz 업그레이드**

- 사업장 1개(본점)만 등록 후 사용
- 나머지 3개 매장도 관리하고 싶은데 Basic은 1개 한도
- **Biz 업그레이드:** 49,900원

- Biz 사용:
  - 4개 매장 모두 등록
  - **멀티 사업장 통합 현황** 테이블 → 매장별 점수 한눈에 비교
    - A점: 71점 (성장기) / B점: 38점 (안정기) / C점: 52점 (성장기) / D점: 21점 (생존기)
  - D점 집중 지원 → FAQ·소개글 키워드 보강
  - 팀 계정 3명 → 각 점장에게 계정 부여, 직접 관리하도록 위임
  - 직원이 매주 리뷰 답변 초안 생성 → 무제한 활용

**3단계: 6개월 후**

- 4개 매장 평균 점수: 45 → 64점
- D점 21 → 58점 (가장 큰 폭 상승)
- 4개 매장 → 5번째 매장 신규 오픈 준비 → 창업 시장 분석 리포트 활용
- **결론:** 사업장 1개당 9,980원 → "직원 한 명 월급보다 싸게 매장 관리"

---

### 4-B. 부정 시나리오 — "Biz 49,900원은 너무 비싸다"

**문제 상황**

- "사업장 4개인데 왜 Pro 4개를 따로 쓰면 안 되나?" 라는 생각
  - Pro × 4 = 91,600원 vs Biz 49,900원 → Biz가 더 싸긴 한데
  - "그냥 Basic 하나로 본점만 관리하면 되지" 결론

- Biz 진입 장벽:
  - 이메일 문의 방식 → 즉시 결제 불가 → 문의 후 회신 대기
  - 회신이 늦어지면 흥미 잃음
  - 토스 즉시 결제가 아닌 협상 과정 번거로움

**이탈 원인**

- "49,900원이면 내가 쓰기엔 너무 크다" → Basic 계속 유지 or 이탈
- 멀티 사업장 가치를 직접 체험하지 못한 상태에서 구매 결정 어려움

**개선 포인트**
- Biz 플랜도 토스페이먼츠 즉시 결제 지원 (현재 이메일 문의 방식)
- "2개 사업장 무료 체험 14일" 같은 Biz 무료 트라이얼 제공
- Basic 사용자가 2번째 사업장 등록 시도 시 → Biz 업그레이드 유도 팝업

---

## 시나리오 5 — 공통 이탈 시나리오

### 5-A. "점수가 오르지 않아서 이탈"

**패턴:**
1. Basic 가입 후 스캔 결과 확인
2. 가이드 내용을 읽었지만 **실행하지 않음**
3. 한 달 후 점수 변화 없음 → "효과 없다"
4. 이탈

**핵심 문제:** 가이드 소비 ≠ 행동 실행

**대응 방안:**
- 행동 완료 체크리스트 (현재 구현됨) → 완료 후 "7일 뒤 결과 알림" 강화
- "이번 주 행동 1가지"를 대시보드 최상단 고정 배너로 노출
- 미완료 행동 → 다음 주 카카오 알림으로 재안내

---

### 5-B. "이미 알고 있던 정보라서 이탈"

**패턴:**
- IT에 밝은 소상공인 → "이 정도 정보는 나도 알아"
- JSON-LD 직접 구현 가능, 키워드 분석도 직접 할 줄 앎
- "굳이 돈 낼 필요 없다"

**핵심 문제:** 숙련 사용자에게 충분한 차별 가치 미제공

**대응 방안:**
- 경쟁사 실시간 변화 알림 (숙련자도 대신 해주기 어려운 자동화)
- AI 인용 실증 데이터 (Gemini 100회 샘플링은 직접 구현이 매우 어려움)

---

### 5-C. "결제 이후 기술적 오류로 이탈"

**예상 오류 목록:**

| 오류 | 원인 | 영향 |
|------|------|------|
| Playwright 스크린샷 타임아웃 | RAM 부족 (vCPU2/4GB) | Before 스크린샷 미캡처 |
| 네이버 AI 브리핑 파싱 실패 | DOM 구조 변경 | Naver 채널 점수 0점 왜곡 |
| 카카오 알림 미발송 | 템플릿 미승인 | Pro 핵심 기능 작동 안 함 |
| Gemini API 할당량 초과 | 구독자 급증 시 | Trial 스캔 실패 |
| Supabase Free Tier 연결 초과 | 동시 사용자 증가 | DB 쿼리 실패 |

**대응 방안:**
- Playwright: 큐 방식 순차 처리 + 타임아웃 시 재시도 1회
- 네이버 파싱: CSS 셀렉터 변경 감지 모니터링 알림
- 카카오 미승인 기간: 이메일 대체 알림
- 구독자 20명 이후: Gemini API 할당량 증설 예약
- 구독자 50명 이후: Supabase Pro Tier 전환

---

### 5-D. "가격 대비 가치 의문으로 이탈"

**패턴:**
- "9,900원이 아깝다" (Basic)
- 매달 자동 결제되는 것을 잊고 있다가 카드 청구서 보고 취소
- "이거 써서 매출이 올랐나?" 측정 불가

**핵심 문제:** ROI 측정 불가 → 구독 지속 명분 약화

**대응 방안 (현재 미구현):**
- Before/After 점수 변화 + "이 기간 동안 AI 브리핑 노출 X회 증가" 월간 성과 요약 카드
- "경쟁사 대비 내 위치" 월간 리포트 이메일 발송
- 취소 버튼 클릭 시 → "지난달 점수 변화 +12점, 정말 취소하시겠어요?" 확인 창

---

## 시나리오 6 — 바이럴·추천 시나리오

### 6-A. 긍정: 자연 추천으로 성장

**경로:**
1. 강남 카페 박민준 → AI 브리핑 노출 성공 → 같은 건물 치킨집에 자랑
2. 치킨집 → 무료 체험 → Basic 가입
3. 마포구 소상공인 커뮤니티에 "네이버 AI 브리핑 잡았다" 후기 업로드
4. 댓글에 "저도 써봤어요" 수십 개 → 유기적 바이럴

**핵심:** 소상공인 커뮤니티(카페24 유저포럼, 소공인 네이버 카페, 배민 사장님 커뮤니티)가 주요 전파 채널

---

### 6-B. 부정: 나쁜 후기가 퍼지는 시나리오

**경로:**
1. 미용실 김지연 → 카카오 알림 안 와서 22,900원 날렸다고 느낌
2. 네이버 카페 "소상공인 정보 공유방" → "AEOlab 사기 아닌가요?" 글 작성
3. 댓글 "저도 효과 없었어요" 동조 → 검색 시 부정 후기 상위 노출

**대응 방안:**
- 이탈 전 불만 접수 채널 (챗봇·이메일)을 눈에 잘 띄게 배치
- 취소 버튼 클릭 시 "무엇이 불편하셨나요?" 빠른 피드백 팝업
- 카카오 알림 미발송 등 기술적 오류 시 → 자동 환불 프로세스

## 시나리오 7 — 구독 생애주기 흐름

```
비로그인 무료 체험 (3회/일)
        │
        ├─ 이탈 (70%) ─────────────────────────→ 재방문 (SNS 후기 보고)
        │
        └─ 회원가입 (30%)
                │
                ├─ 무료 플랜 (가입만) ─ 이탈 (80%)
                │
                └─ Basic 가입 (20%)
                        │
                        ├─ 1개월 이탈 (40%) ← 행동 안 함, 효과 체감 못함
                        │
                        ├─ 3개월 지속 (40%) ← 점수 상승 체감, 루틴화
                        │       │
                        │       └─ Pro 업그레이드 (20%) ← 경쟁 심화, 더 많은 기능 필요
                        │               │
                        │               └─ Biz 업그레이드 (5%) ← 다점포 확장
                        │
                        └─ 창업 패키지 (10%) ← 창업 준비 목적
                                │
                                └─ 창업 후 Basic 전환 (60%)
```
