# AEOlab 개선 방안 v1.1 — 소상공인 관점 피드백

> **v1.0에서 수정된 핵심 방향 2가지:**  
> 1. "손님 수" 숫자 추정 제거 → 실제 측정 가능한 AI 노출 데이터로 대체  
> 2. "무엇을 해야 하는지"에서 "어떻게 하는지 단계별"로 전환  
> **작성 기준:** 2026-04-14

---

## 먼저: 왜 "손님 수 추정"을 쓰면 안 되는가

> "이번 주 AI에서 내 가게 본 손님 약 23명"

이 표현의 문제:
- AI 노출 = 실제 방문으로 이어지지 않는 경우가 훨씬 많음
- "23명"이 실제와 다르면 소상공인이 서비스를 불신함
- 과장 광고로 오해받을 법적 위험

**대신 쓸 수 있는 것: AEOlab이 실제로 측정하는 수치**

| 기존 (추정, 사용 불가) | 대체 (실제 측정값, 사용 가능) |
|---------------------|--------------------------|
| "손님 약 23명" | "Gemini에 100번 물었을 때 내 가게가 나온 횟수: 23번" |
| "광고 효과 168,000원 절약" | "이번 주 AI 노출 빈도: 23% (지난주 19%에서 +4%p)" |
| "예약 2명 증가 예상" | "경쟁 카페 5곳 중 AI 노출 순위: 3위 (지난주 5위)" |

숫자는 그대로 쓰되, **"AI가 실제로 내 가게를 언급한 횟수"** 라고 표현하면 정확하고 신뢰할 수 있다.

## Part 1. 소상공인에게 의미 있는 지표 재설계

### 현재 대시보드가 보여주는 것

```
Track 1 점수: 41점
Track 2 점수: 28점
통합 점수: 37점
성장 단계: 안정기
```

소상공인의 반응: "이게 뭔 소리야?"

---

### 개선안: 점수를 "AI가 실제로 한 일"로 번역

**변경 전 (추상적 점수)**
```
Track 1 점수: 41점
```

**변경 후 (실제 측정값)**
```
네이버 AI 브리핑 이번 주 결과

Gemini에게 100번 물었을 때 내 가게가 나온 횟수
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
이번 주  ████████░░░░░░░░░░  23번
지난 주  ██████░░░░░░░░░░░░  19번  (4번 증가 ↑)
경쟁 1위 ████████████████░░  41번

네이버 AI 브리핑  이번 주 노출됨 ✓
ChatGPT          이번 주 미노출
Perplexity       이번 주 노출됨 ✓
Google AI        이번 주 미노출
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

이 숫자들은 **AEOlab이 실제로 측정한 값**이다. 추정이 아니다.  
소상공인이 이해할 수 있고, 거짓이 없다.

---

### 경쟁사 비교: "몇 위"로 표현

```
강남구 카페 AI 노출 순위 (이번 주)

1위 OO카페    ████████████████  41번
2위 △△카페   ████████████░░░░  31번
3위 내 가게   ████████░░░░░░░░  23번 ← 나
4위 □□카페   █████░░░░░░░░░░░  14번
5위 ◇◇카페   ███░░░░░░░░░░░░░   9번

→ 지난주 5위에서 이번주 3위로 올랐습니다.
→ 1위와의 차이: 18번. 좁히려면 FAQ 추가가 가장 효과적입니다.
```

"순위"는 소상공인이 가장 잘 이해하는 개념이다.  
배달앱 순위, 네이버 지도 순위 — 이미 익숙한 언어다.

### 월간 성과 리포트: "내가 한 행동"과 "AI의 반응" 연결

**결제일 3일 전 카카오 발송**

```
박민준 사장님, 이번 달 강남 카페 AI 노출 요약입니다.

─────────────────────────────────
AI가 내 가게를 언급한 횟수 변화:
  이번 달: 월평균 23번/주
  지난 달: 월평균 14번/주
  변화: +9번/주 증가 ↑

내가 한 행동 → AI의 반응:
  ✓ FAQ 3개 등록 → Gemini 노출 +6번/주 증가
  ✓ 소식 2회 업데이트 → 최신성 점수 상승
  ○ 리뷰 답변 미실행 → 이 부분은 아직 기회

경쟁사 순위: 5위 → 3위
─────────────────────────────────

다음 달 1위를 노리려면:
"리뷰에 '강남 브런치' 키워드 포함 답변"이 필요합니다.
→ [지금 답변 초안 생성 →]
```

추정 수치 없음. 모두 실제 측정값이다.

## Part 2. "어떻게 하는지" 단계별 가이드 — 핵심 개선

### 현재 가이드의 문제

```
[현재 가이드]
"스마트플레이스 FAQ에 키워드를 등록하세요.
AI 브리핑은 FAQ 형식의 텍스트를 인용할 확률이
가장 높습니다. 업종에 맞는 키워드를..."
```

소상공인의 반응:
- "스마트플레이스 FAQ가 어디 있지?"
- "PC로 해야 해? 핸드폰으로 해?"
- "어떤 메뉴 눌러야 하지?"
- "뭐라고 쓰면 되는 거야?"

**가이드를 읽게 만들 것이 아니라, 화면을 보면서 따라하게 만들어야 한다.**

---

### 핵심 행동 4가지 — 단계별 실행 가이드

---

#### 행동 1. 스마트플레이스 FAQ 등록하기

> **효과:** Gemini·네이버 AI 브리핑이 FAQ를 직접 인용함  
> **소요 시간:** 5분  
> **필요한 것:** 스마트폰 또는 PC

---

**[스마트폰으로 하는 방법]**

```

       → 또는 아래 버튼 클릭:
       [스마트플레이스 앱 열기 →]

2단계: 내 업체 선택
       → 로그인 후 내 가게 이름 클릭

3단계: "소식·Q&A" 탭 선택
       → 화면 상단 탭 중 "Q&A" 클릭

4단계: 아래 문구를 그대로 복사해서 붙여넣기
       ↓↓↓

┌─────────────────────────────────────┐
│ Q. 주차가 가능한가요?                │
│                                     │
│ A. 네, 가게 앞 공영주차장을          │
│    이용하실 수 있습니다.             │
│    강남 브런치 카페로 오시는 분들께  │

│                          [복사] ✓  │

5단계: "등록" 버튼 클릭
       → 완료!

6단계: 여기를 눌러 완료 표시
       [✓ FAQ 등록 완료 — 효과 측정 시작]

## Part 3. 가이드 화면 구성 방식 개선

### 현재 가이드 페이지 레이아웃의 문제

```
[현재 순서]
긴 텍스트 설명 → 복사 버튼 → 완료 체크

[소상공인의 행동 패턴]
긴 텍스트 → (읽다가 이탈) → 복사 버튼 도달 못함
```

---

### 개선: "행동 먼저, 설명은 접기"

```
[개선된 순서]

오늘 할 것 1가지                        [왜 해야 하나 ▼]
─────────────────────────────────────
FAQ 등록 (5분)

지금 내 가게에 없는 키워드: "강남 브런치"

[1] 아래 문구 복사
    ┌──────────────────────┐
    │ Q. 주차가...         │  [복사]
    └──────────────────────┘

[2] 스마트플레이스 앱 열기
    [앱 열기 →]

[3] Q&A 탭 → 붙여넣기 → 저장

[✓ 완료했어요]  ← 클릭하면 다음 행동으로
─────────────────────────────────────

[왜 해야 하나 ▼] 를 누르면:
"AI 브리핑은 FAQ 형식 텍스트를 가장 많이 인용합니다.
 FAQ를 등록하면 7~14일 내 Gemini 노출이 증가합니다."
```

**원칙:**
- 설명은 기본적으로 숨김 (접기/펼치기)
- 행동 → 복사 → 앱 열기 → 완료 체크 순서가 항상 먼저
- "왜?"가 궁금한 사람만 설명 읽음

## Part 4. 플랜별 "어떻게" 가이드 범위

### Basic 플랜 — "스마트플레이스 3가지"

Basic 사용자가 배워야 할 것은 단 3가지다.

```
Week 1: FAQ 등록하기 (위 행동 1 가이드)
Week 2: 소개글 키워드 추가하기 (위 행동 2 가이드)
Week 3: 리뷰 답변 달기 (위 행동 3 가이드)
Week 4: 효과 확인 + 소식 올리기 (위 행동 4 가이드)
```

이 4가지를 완료하면 Track 1 점수는 반드시 오른다.  
4가지 가이드를 순서대로 보여주는 **"30일 플랜"** 이 온보딩의 핵심이다.

---

### Pro 플랜 — "경쟁사 대응"

Pro 사용자에게 추가로 필요한 "어떻게":

```
경쟁사가 FAQ를 추가했다는 알림을 받았을 때:

알림: "A카페가 '강남 탈색' 키워드를 FAQ에 추가했습니다"

→ [바로 대응하기]

1단계: 아래 대응 FAQ 복사
       ┌──────────────────────────────┐
       │ Q. 강남에서 탈색 잘하는      │
       │    곳 추천해주세요           │
       │ A. 저희 카페에서 커피 한 잔  │
       │    하시면서 잠깐 쉬어가세요. │
       │    강남 탈색 후 논현동 브런치│
       │    로 많이 오십니다!         │
       │                  [복사] ✓  │
       └──────────────────────────────┘
