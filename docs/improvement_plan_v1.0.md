# AEOlab 개선 방안 — 매출·광고 중심 소상공인 관점 피드백 v1.0

> **작성 관점:** "네이버에 광고비 달마다 30만원 내고 있는데, 이게 더 효과 있나?"를 묻는 소상공인  
> **핵심 질문:** 이 서비스가 내 매출에 직접 연결되는가?  
> **작성 기준:** 2026-04-14

---

## 소상공인의 실제 머릿속

서비스를 개선하기 전에 소상공인이 실제로 어떻게 생각하는지 먼저 직시해야 한다.

```
[소상공인의 하루]

오전: 재료 준비, 직원 관리, 청소
점심: 손님 응대, 주방
오후: 발주, 회계, 배달앱 주문 처리
저녁: 마감, 피로
밤: 스마트폰으로 리뷰 확인, 경쟁 가게 염탐

→ AEOlab을 볼 수 있는 시간: 하루 5분
→ 이 5분에 "내 매출에 도움이 됐나?"를 확인해야 한다
```

**소상공인이 진짜로 원하는 것 3가지**

1. **"오늘 손님이 더 왔으면 좋겠다"** — 즉각적 효과
2. **"광고비 낭비하고 싶지 않다"** — 비용 효율
3. **"경쟁 가게보다 앞서고 싶다"** — 상대적 우위

현재 AEOlab이 제공하는 것: **"AI 브리핑 노출 점수"**  
소상공인이 실제로 원하는 것: **"손님이 늘었는가"**

이 간격을 줄이는 것이 개선의 핵심이다.

---

## 1. 가장 큰 문제 — "점수가 매출과 연결되는 느낌이 없다"

### 현재 상황

소상공인이 AEOlab을 켜면 이런 화면을 본다.

```
Track 1 점수: 41점
Track 2 점수: 28점  
통합 점수: 37점
성장 단계: 안정기
```

**소상공인의 반응:** "그래서 뭐? 이게 손님이랑 무슨 상관이야?"

점수 자체는 의미가 없다. 소상공인에게 의미 있는 숫자는 하나뿐이다.

> **"오늘 AI 검색에서 내 가게를 찾아온 예상 손님 수"**

---

### 개선 방안 1: 점수 → 손님 수 번역 시스템

**"AI 브리핑 노출 = 손님 몇 명"으로 바꿔 보여주기**

현재 네이버 AI 브리핑 공식 데이터:
- AI 브리핑 CTR: +27.4%
- 예약·주문 전환율: +8% (2025년 8월 네이버 공식)

이 데이터를 활용하면 다음과 같이 표현 가능하다.

```
[현재 표시]                    [개선 후 표시]
Track 1 점수: 41점    →     이번 주 AI에서 내 가게 본 사람: 약 23명
                             → 이 중 예약·방문 예상: 약 2명
                             
                             지난주보다 +5명 증가 ↑
                             경쟁 상위 가게: 약 67명
```

**구현 방법:**
- Gemini 100회 샘플링의 `exposure_freq`를 기반으로
- 네이버 AI 브리핑 검색량 추정치 × 노출 빈도 × CTR로 역산
- "약 X명" 표현 (정확한 수치가 아님을 명시)
- 주간 비교: 지난주 대비 증감 표시

**소상공인 입장 효과:**
- "점수 41점"보다 "손님 2명 더"가 훨씬 와닿음
- 매주 증가하는 숫자를 보면 서비스 지속 이유가 생김

---

### 개선 방안 2: 광고비 비교 배너

**"네이버 광고 vs AEOlab" 직접 비교**

소상공인이 가장 많이 쓰는 마케팅은 **네이버 플레이스 광고 (월 10~50만원)** 다.

```
[대시보드 상단 배너]

📊 이번 달 AI 검색 노출 분석

네이버 광고 없이 AI 브리핑으로 노출된 횟수: 약 89회
광고로 같은 노출을 얻으려면: 약 178,000원 (클릭당 2,000원 기준)

→ AEOlab 월 구독료: 9,900원
→ 이번 달 절약 추정: 약 168,100원
```

**주의사항:**
- "추정", "약" 표현을 반드시 사용하여 과장 광고 방지
- 클릭당 단가는 업종별로 다르므로 업종별 평균값 사용
- 법적 검토 후 표현 조정

**소상공인 입장 효과:**
- 9,900원 구독료가 "비싸다"에서 "이득"으로 인식 전환
- 구독 취소율 대폭 감소 예상

## 2. "가이드는 읽었는데 어떻게 하는지 모르겠다"

### 현재 문제

현재 가이드 페이지는 **"무엇을 해야 하는지"** 는 잘 설명하지만,  
**"지금 이 화면에서 어떻게 실행하는지"** 는 알려주지 않는다.

```
[현재 가이드 내용]
"스마트플레이스 FAQ에 키워드를 추가하세요"
"AI 브리핑 경로 B: FAQ 등록이 가장 효과적입니다"

[소상공인의 생각]
"스마트플레이스 FAQ가 어디 있지?"
"PC로 해야 하나, 폰으로 해야 하나?"
"뭐라고 쓰면 되지?"
```

---

### 개선 방안 3: "5분 실행 카드" 시스템

**가이드를 읽는 것이 아니라 실행하는 것으로 전환**

```
[현재]
─────────────────────────────
AI 브리핑 경로 B: FAQ 등록
왜 효과적인가: AI 브리핑은 FAQ 형식의 텍스트를 
인용할 확률이 가장 높습니다...
(이하 300자 설명)
─────────────────────────────

[개선 후]
─────────────────────────────
🔥 오늘 5분 — 지금 바로 복사해서 붙여넣기

1단계: 아래 문구를 복사하세요
┌─────────────────────────────┐
│ Q. 주차가 가능한가요?        │
│ A. 네, 강남역 3번 출구 앞    │
│ 공영주차장 이용 가능합니다.  │
│ 홍대 브런치 카페로 가장      │
│ 가까운 주차 공간입니다.      │
│                    [복사] ✓ │
└─────────────────────────────┘

2단계: 스마트플레이스 앱 열기
[스마트플레이스 앱 열기 →]  ← 딥링크

3단계: 소식/Q&A 탭 → 붙여넣기
완료 후 여기를 눌러주세요: [✓ 완료]
─────────────────────────────
```

**핵심 원칙:**
- 설명 최소화, 실행 최대화
- 복사 버튼 1번 클릭 → 스마트플레이스 앱 열기 → 붙여넣기
- 완료 체크 → "7일 후 변화를 알려드립니다" 카운트다운 시작

---

### 개선 방안 4: 스마트플레이스 앱 딥링크 연동

**가장 큰 장벽: "스마트플레이스 어디서 열어요?"**

현재: 사용자가 직접 스마트플레이스 앱을 찾아서 열어야 함  
개선: AEOlab에서 버튼 한 번으로 스마트플레이스 해당 메뉴로 이동

```
[모바일 화면]

FAQ 문구 복사 완료 ✓

다음 단계:
[스마트플레이스 앱에서 Q&A 등록하기 →]
  ↳ naver://smartplace/qna 딥링크

앱이 없으시면:
[스마트플레이스 웹에서 열기 →]
  ↳ business.naver.com 직접 링크
```

**구현 난이도:** 낮음 (딥링크 URL 확인 후 `<a href>` 처리)  
**효과:** 실행 전환율 3배 이상 증가 예상

## 3. "광고를 끊고 이걸로 대체할 수 있나?"

### 소상공인의 핵심 고민

매달 네이버 플레이스 광고에 20~50만원 지출.  
AEOlab을 쓰면 광고비를 줄일 수 있는가? — 이 질문에 답하지 못하면 이탈한다.

---

### 개선 방안 5: 광고 대체 시뮬레이터

**"광고 예산을 AEOlab으로 얼마나 줄일 수 있나" 계산기**

```
[설정 페이지 또는 가이드 페이지 내]

📊 광고 최적화 시뮬레이터

현재 광고비: [___]만원/월  ← 사용자 입력
업종: 카페
지역: 강남

─────────────────────────────────
분석 결과:

현재 광고 방식
 → 네이버 플레이스 광고 클릭 단가: 약 1,500~3,000원
 → 월 방문 예상: 약 150~300명 (광고비 대비)

AEOlab AI 노출 방식 (현재 내 점수 41점 기준)
 → AI 브리핑 노출 시 클릭 비용: 0원
 → 점수 70점 달성 시 월 예상 노출: 약 200~400명

추천: 
 광고비 30% 줄이고 → AI 최적화에 집중하면
 같은 효과를 9,900원으로 유지 가능

[AI 노출 70점 달성 가이드 보기 →]
─────────────────────────────────
```

**주의:** 추정치 명시, 법적 과장 표현 주의  
**효과:** "광고 대체재"로 포지셔닝 강화 → 구독 당위성 확보

---

### 개선 방안 6: 네이버 플레이스 광고와 연계 분석

**"광고 클릭 vs AI 브리핑 자연 유입" 비교 대시보드**

소상공인이 네이버 광고를 켜놓고 AEOlab을 쓰는 경우,  
두 채널의 효과를 비교해서 보여주면 광고비 최적화에 직접 도움이 된다.

```
[대시보드 새 카드: 트래픽 채널 분석]

이번 주 내 가게를 찾은 경로 추정:

AI 브리핑 자연 노출  ████████░░  67%  약 34명
네이버 광고          ████░░░░░░  33%  약 17명
기타                 ░░░░░░░░░░   0%

→ 자연 노출이 광고보다 2배 많습니다.
→ 광고비를 20% 줄여도 전체 유입은 유지될 수 있습니다.
```

**구현 방법:**
- Gemini 샘플링 노출 빈도 기반 자연 유입 추정
- 광고 데이터는 사용자가 직접 입력 (네이버 API 연동은 별도 과제)

## 4. "리뷰 관리가 제일 급한데"

### 소상공인의 실제 우선순위

**AI 브리핑 노출보다 리뷰 평점 관리가 더 급하다고 느끼는 소상공인이 많다.**

이유:
- 별점 1개 리뷰 1개 → 예약 취소 3~5건 발생
- "리뷰 답변을 잘 달면 AI가 긍정적으로 인용한다"는 사실을 모름

현재 AEOlab의 리뷰 관련 기능:
- 리뷰 답변 초안 (월 10~무제한)
- 리뷰 감정 분석 대시보드
- 리뷰 키워드 갭 분석

**문제:** 이 기능들이 "리뷰 관리"라는 프레임으로 묶여있지 않고 각각 흩어져 있다.

---

### 개선 방안 7: "리뷰 관리 전용 탭" 신설

**소상공인이 가장 자주 하는 일을 별도 탭으로**

```
[사이드바 메뉴]

현재:                     개선:
─ 대시보드               ─ 대시보드
─ 경쟁사                 ─ 리뷰 관리  ← NEW (소상공인 1순위)
─ 개선 가이드             ─ 경쟁사
─ 히스토리               ─ 개선 가이드
─ 스키마                 ─ 히스토리
─ 설정                   ─ 설정
```

**리뷰 관리 탭 구성:**
```
1. 감정 분석 현황
   긍정 73% | 중립 18% | 부정 9%
   
2. 최근 부정 리뷰 (요주의)
   ⚠️ "주차가 불편해요" — 3일 전
   [답변 초안 생성 →]
   
3. 리뷰 유도 전략
   이번 달 리뷰 수: 12개
   경쟁사 평균: 34개
   
   [리뷰 유도 QR 문구 복사 →]
   목표 키워드: "홍대 브런치 맛집"
   
4. AI가 인용한 리뷰
   "음식이 정말 맛있어요" — Gemini에서 인용됨 ✓
   "분위기가 좋아요" — 네이버 AI에서 인용됨 ✓
```

**소상공인 입장 효과:**
- "리뷰 관리"라는 친숙한 프레임으로 진입 → AI 최적화 자연스럽게 연결
- 부정 리뷰 → 즉시 답변 초안 → 키워드 포함 답변으로 AI 개선 효과 이중 획득

## 5. "경쟁 가게가 뭘 바꿨는지 빨리 알고 싶다"

### 현재 문제

경쟁사 변화 알림은 Pro+ 전용이고, 카카오 알림톡 미승인 상태.  
소상공인 입장에서 가장 원하는 기능인데 가장 불안정한 기능이다.

---

### 개선 방안 8: 경쟁사 변화 "주간 요약 카드"

**카카오 알림 없이도 경쟁사 변화를 즉시 파악하는 방법**

```
[매주 월요일 대시보드 상단 팝업]

📍 이번 주 경쟁사 변화 요약

홍대 A미용실     ▲ 점수 +8점 (FAQ 새로 추가한 것으로 추정)
마포 B미용실     ─ 변화 없음
연남 C미용실     ▼ 점수 -3점 (최신 소식 미업데이트)

→ A미용실이 이번 주 "홍대 탈색 매직" 키워드를 추가했습니다
   [내 가게에도 추가하기 →]  ← 즉시 실행 연결
```

**핵심:** 알림으로 끝나는 것이 아니라 **즉시 대응 행동으로 연결**  
경쟁사가 뭔가를 했다 → 나도 바로 할 수 있는 버튼 제공

---

### 개선 방안 9: "경쟁사 베끼기 기능" (긍정적 의미로)

**경쟁사가 쓰는 키워드를 1클릭으로 내 가게에 적용**

```
[경쟁사 분석 화면]

홍대 A미용실이 사용 중인 키워드:
  ✓ 홍대 탈색     [내 소개글에 추가 →]
  ✓ 마포 펌        [내 FAQ에 추가 →]
  ✓ 홍대 매직스트레이트  [내 소식에 추가 →]

내가 아직 안 쓰는 키워드: 3개
[전부 적용 초안 생성 →]
```

소상공인의 언어: "저 가게가 왜 잘 되는지 따라해봐야지"  
기술적 언어: "경쟁사 키워드 갭 분석 후 최적화"  
→ 같은 것이지만 소상공인 언어로 표현하면 이해와 실행이 빠름

---

## 6. "월 9,900원 낼 때마다 효과 있나 확인하고 싶다"

### 현재 문제

구독 갱신일이 다가올 때 소상공인은 이렇게 생각한다.

> "지난달에 9,900원 냈는데 뭔가 달라졌나? 모르겠는데. 그냥 취소할까?"

현재 AEOlab에는 **"지난달 대비 성과 요약"을 자동으로 보내주는 기능이 없다.**

---

### 개선 방안 10: 월간 성과 리포트 자동 발송

**결제일 3일 전 카카오 or 이메일로 발송**

## 8. "처음 쓸 때 뭘 해야 할지 모르겠다" — 온보딩 개선

```

  "첫 번째 행동 가이드 (5분)" →
  첫 성과 확인 (1주일 후) →
  두 번째 행동 →
  루틴 형성
```

### 개선 방안 12: 30일 온보딩 플로우

  ○ 이번 주: 소식 1개 올리기 (5분)

2주차 목표: 경쟁사 파악
  ○ 경쟁 카페 3곳 등록 + 갭 확인

  ○ 최근 리뷰 5개에 키워드 답변

4주차 목표: 효과 확인
  ○ 점수 변화 확인 + 다음 달 계획

──────────────────────────────

### 개선 방안 13: "광고 키워드 → AI 최적화" 연계 도구

**네이버 광고에서 효과 있는 키워드를 AI 최적화에 바로 적용**

```

📊 네이버 광고 키워드 연계

광고에서 클릭 많이 받은 키워드를 입력하면
AI 최적화에 바로 적용해드립니다.

내 광고 클릭 키워드 (직접 입력):
[강남 카페          ] [홍대 브런치         ] [논현 커피           ]

→ 분석 결과:
  "강남 카페" — 이미 AI에 73% 노출 중 (광고 없어도 잘 됨)
  "홍대 브런치" — AI 노출 12% (광고 의존 높음, 최적화 필요)
  "논현 커피" — AI 노출 0% (지금 당장 FAQ에 추가하세요)

[AI 최적화 우선순위 키워드 가이드 보기 →]
```

**소상공인 입장 효과:**
- 광고비 낭비 키워드 파악 → 비용 절감
- AI 최적화 필요 키워드 명확화 → 행동 동기
