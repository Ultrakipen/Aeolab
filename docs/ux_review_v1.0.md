## 2. 구간별 상세 점검

### 2-1. 무료체험 (trial/page.tsx)

#### ✅ 수정 완료 사항

1. **TRIAL_DAY_LIMIT 운영값 복구**
   - 변경: `20` → `3` (하루 3회 제한)
   - 이유: 운영 환경에서 하루 20번 체험 가능했던 보안 버그

2. **sticky 배너 문구 수정**
   - 변경 전: "이 분석을 저장하고 매주 자동 진단 받으려면?"
   - 변경 후: "이 결과를 기반으로 매주 자동 진단받고"
   - 이유: 실제로 체험 결과가 저장되지 않아 가입 후 배신감 유발

3. **STEP 1 서비스 설명 추가**
   - 추가: "AI가 내 가게를 알고 있는지 무료로 확인해 드립니다"
   - 이유: 처음 방문한 사장님이 3초 안에 목적 파악 가능하도록

#### 남은 과제 (중기)

- **체험 결과 신뢰성 안내 부재**: 10회 샘플링 vs 정식 100회 차이를 결과 화면에서 설명 없음
  - 권장: 결과 상단에 "이 결과는 빠른 테스트(10회) 기준입니다. 정식 스캔은 100회 반복 분석합니다" 배너
- **스마트플레이스 체크박스 맥락 설명 부재**: FAQ/최신글/소개 등록 여부를 묻는 이유 설명 없음
  - 권장: 각 체크박스 옆 `?` 아이콘 + 툴팁 "이걸 등록하면 AI 브리핑 노출이 올라가요"
- **쿨다운 시 대안 경로 없음**: 회원가입 유도만 있고 "내일 다시" 안내 없음

### 2-2. 요금제 안내 (pricing/page.tsx)

#### 확인 결과

- "14일 무료 체험" 문구: **미존재** (FAQ에 "3~14일"은 AI 인식 소요 기간으로 정상)
- Biz CTA: **ContactModal로 이미 구현됨** (mailto 아님)
- 비교표 overflow: **별도 확인 필요** — 모바일에서 4열 비교표 가로 스크롤 처리 여부

#### 남은 과제 (중기)

- **Basic vs 창업패키지 차이 즉시 이해 어려움**:
  - Basic: 기존 사업장 최적화
  - 창업패키지: 창업 전 시장조사 + 경쟁 분석
  - 권장: 플랜 선택 전 "나는 이런 사람이에요" 2~3개 버튼 (현재 사업자/예비 창업자/시장 조사)
- **스캔 횟수·경쟁사 수 한도가 비교표에 숫자로만 표시**:
  - "월 4회 스캔" → "매주 1번 자동으로 점수 확인"처럼 체감 언어 필요

### 2-3. 대시보드 (dashboard/page.tsx + DualTrackCard.tsx)

#### 현재 상태

- Empty state: 사업장 미등록 시 7개 AI 목록 + 스캔 안내 → 소상공인 친화적 ✅
- DualTrackCard: Lucide 아이콘(`Sprout`, `TrendingUp`, `Flame`, `Trophy`)으로 성장단계 표시 ✅
- 3단 행동 카드(오늘/이번 주/이달) 자동 도출 ✅

#### 남은 과제 (중기)

- **ScoreBar sublabel이 개발자 언어**:
  - 현재: "스마트플레이스 완성도·리뷰 키워드·FAQ 등록 여부"
  - 권장: "이 점수가 낮으면 네이버 AI가 내 가게를 잘 모릅니다"
- **업종 평균 비교 문구 text-xs로 너무 작음**:
  - "▲ 업종 평균보다 5점 높음" → `text-sm` 이상으로 확대 필요
  - 이 정보가 사장님에게 가장 동기부여가 됨
- **없는 키워드 amber 박스 → 가이드 연결 시 키워드 파라미터 미전달**:
  - 현재: `/guide`로만 이동
  - 권장: `/guide?keyword=xxx`로 이동하여 가이드에서 해당 키워드 섹션 강조

#### 남은 과제 (중기)

  - 현재: 4번째 경로 전체 잠금
  - 권장: 레이블("소개글 수정 · 1회, 영구 효과") + 복사 문구만 Pro 전용으로 변경

  - 처음 방문 시 "이 가이드가 무엇인지" 설명이 없음

## 4. 소상공인이 느끼는 현실적 여정

### 긍정 여정 (잘 되는 경우)
```
1. 네이버 검색 "AI 검색 최적화" → aeolab.co.kr 랜딩
2. 업종(카페)/지역(강남) 입력 → 무료체험 시작
3. 결과: "네이버 AI 브리핑 점수 38점" + "없는 키워드: 아메리카노, 강남역 카페"
   → "아 내가 네이버에서 잘 안 나오는 이유가 이거구나!" (AHA 순간)
4. 회원가입 → Basic 결제 → 대시보드 확인
5. 가이드 → "FAQ 등록하기" 복사 버튼 클릭 → 스마트플레이스에 붙여넣기
6. 1주일 후 "점수가 올랐어요!" 카카오 알림
```

### 부정 여정 (이탈하는 경우)
```
1. 무료체험 결과: "unified_score: 38" (설명 없는 숫자)
   → "이게 높은 건지 낮은 건지 모름" → 이탈
2. 가이드 페이지 접속 → 가이드 없음 상태 → 생성 버튼 클릭 → 30초 대기 → 빈 화면
   → "고장났나?" → 이탈
3. 경쟁사 추가 → 아무것도 안 보임 (다음 스캔 전) → "기능이 없네" → 이탈
4. 없는 키워드 클릭 → 가이드 이동 → 해당 키워드 관련 내용 찾기 어려움 → 이탈
```
